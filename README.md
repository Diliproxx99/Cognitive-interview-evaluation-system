 Cognitive Interview Evaluation System

📌 Description

The AI Cognitive Interview Evaluation System is an intelligent web-based platform designed to evaluate candidate performance during interviews using Artificial Intelligence (AI) and Natural Language Processing (NLP). The system supports multiple input modes including text, voice, and video-based responses, enabling a more comprehensive and realistic interview evaluation process.

It analyzes candidate responses to assess key parameters such as clarity, relevance, confidence, communication skills, and sentiment. The system provides automated scoring along with detailed feedback, reducing human bias and improving consistency in the hiring process.

---

🚀 Features

- Text-based interview evaluation
- Voice-based response support (speech-to-text / text-to-speech)
- Video-based interaction support
- Resume upload functionality
- Automated performance scoring
- Sentiment and communication analysis
- Result generation with feedback
- User-based system with role management (Candidate/Admin)

---

🛠️ Tech Stack

- Python (Django Framework)
- Natural Language Processing (NLP)
- HTML, CSS, Bootstrap
- JavaScript
- SQLite Database

---

⚙️ How It Works

1. User registers and logs into the system
2. Candidate uploads resume
3. System presents interview questions
4. Candidate responds via text, voice, or video
5. AI analyzes responses using NLP techniques
6. System generates score and feedback
7. Results are displayed to user/admin

---

▶️ How to Run

1. Clone the repository
2. Install dependencies
3. Run the server:
   python manage.py runserver
4. Open browser and go to:
   http://127.0.0.1:8000/
